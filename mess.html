How to Use DramaPigeon

Receive Your Code

When someone sends you a message through DramaPigeon, you'll receive a unique access code via Instagram DM.

Click the Envelope

Click on the envelope above to bring up the code input dialog.

Enter Your Code

Type in your access code to unlock and view your message.

Reply if You Wish

After reading your message, you can choose to send an anonymous reply (or with your name) by clicking the "Send a Reply" button.
